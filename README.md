# Momento

## Backend Step by Step Procedure

### Install dependencies

```
mvn clean install
```

### Run backend server (http://localhost:8080)

```
mvn spring-boot:run
```

## UI local Setup

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```
